# Security Policy

## Supported Versions

This project was developed for experimental and study purposes. As it is an experimental project, only the latest version is maintained.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you identify any security vulnerability or technical flaw, please do not open a public issue. To maintain the integrity of the experiment, report the issue by following these steps:

1. Send an email to: **devmayy@gmail.com**
2. In the email body, describe the vulnerability and, if possible, the steps to reproduce it.

Thank you for helping to make this project more secure!

## Privacy and Data Usage

To ensure the transparency of this experiment:
* **Collection:** The application exclusively requests the user's **Name** and **email**.
* **Purpose:** This data is used only for sending automatic system notifications.
* **Treatment:** The data is not shared with third parties and is used strictly within the scope of this project.

# Política de Segurança

## Versões Suportadas

Este é um projeto desenvolvido para fins de experimento e estudo. Por ser um projeto experimental, apenas a versão mais recente é mantida.

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Relatando uma Vulnerabilidade

Se você identificar qualquer vulnerabilidade de segurança ou falha técnica, por favor, não abra uma *issue* pública. Para manter a integridade do experimento, relate o problema seguindo estas etapas:

1. Envie um e-mail para: **devmayy@gmail.com**
2. No corpo do e-mail, descreva a vulnerabilidade e, se possível, os passos para reproduzi-la.

Obrigado por ajudar a tornar este projeto mais seguro!

## Privacidade e Uso de Dados

Para garantir a transparência deste experimento:
* **Coleta:** O aplicativo solicita exclusivamente o **Nome** e o **e-mail** do usuário.
* **Finalidade:** Este dado é utilizado apenas para o envio de notificações automáticas do sistema.
* **Tratamento:** Os dados não são compartilhados com terceiros e são utilizados estritamente dentro do escopo deste projeto.
