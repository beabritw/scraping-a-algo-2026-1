# Scraping - Assistente de Lances

Guia rápido para rodar o projeto.

## 1. Preparar Ambiente
No terminal, na raiz do projeto, execute:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

```


Crie um .env com o conteudo:

```bash
EMAIL_SENHA=abcd efgh ijkl mnop
```
O sistema exige uma "Senha de App" do Google para enviar notificações. Gere a sua no painel do Google e exporte no terminal.
